Código-fonte/Pré-formatação

O comando document.getElementById('teste')é escrito em linguagem JavaScript, para sair bem formatado o código uso a tag Code que deixa a fonte monoespassada, ótima para ler códigos.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Acima temos um código utilizando somente a tag code, ele certamente não está formatado assim na ide de desenvolvimento, observe abaixo a diferença de quando envelopamos a tag code com a tag pre (esta tag respeita a identação e formatação)

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Um comando importante usado para deslocar o código corretamente na tela é:
Seleciono todo o texto do código, shift + tab
Dessa forma posso deslocar para a esquerda, para a direita seleciono e aperto o tab
